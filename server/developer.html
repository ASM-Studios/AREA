Project Documentation for Devs

Introduction

This documentation is intended for developers working on the project. It explains the core concepts, including the use of RabbitMQ for RabbitMQ: Example Consumer, the Backend Structure structure (controllers, middleware, models, etc.), and how to create and manage consumers. The backend uses MariaDB for data storage.
